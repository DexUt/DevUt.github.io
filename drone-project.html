Drone Project

Autonomous quadrotor stack built in PX4 + ROS + Gazebo (SITL): takeoff, PD trajectory tracking, and real-time LiDAR-based obstacle avoidance via Artificial Potential Fields.

Methods

  • PX4 SITL with Gazebo; MAVROS offboard control; C++ ROS nodes streaming position/velocity setpoints (>2 Hz); PD outer-loop for trajectory tracking.
  • Repulsive-only Artificial Potential Field for reactive avoidance using 2D LiDAR; returns to hover when clear.

Tools

  • PX4 Autopilot, MAVROS, Gazebo Classic, ROS (C++), EKF2 in PX4, MATLAB (APF prototyping).

Outcome

  • Autonomous takeoff/hover at 2 m; stable circular-path tracking; real-time avoidance when a moving obstacle enters the influence radius; modular architecture ready to import the Manta geometry for HIL/field work.
